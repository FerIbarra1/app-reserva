# Practica de Booking con MERN React - MongoDB - Node - Express


## Correr Localmente

### 1. Crear Archivo .env

- Duplicar .env.example en la carpeta backend y cambiarle el nombre a .env

### 2. Correr Backend

```
$ cd api
$ npm install
$ npm start
```

### 3. Correr Frontend

```
# abrir nueva terminal
$ cd app-reserva
$ yarn add
$ yarn dev
```
